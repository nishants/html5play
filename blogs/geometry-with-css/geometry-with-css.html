Drawing square is the simplest. Just create an element with fixed height and width.Then set a background color.
.css-shape{ background-color: green; display: inline; } .square{ width: 100px; height: 100px; }
An border becomes semicircle, if radius of the edge is half its length. So a square becomes circle, if border radius is laf the length. 
.circle{
    width: 100px;
    height: 100px;
    border-radius: 50%;
}
The rectangle with border radius 50%(for both height and width) is an oval 
.rectangle{
    width: 200px;
    height: 100px;
    border-radius: 50%;
}
